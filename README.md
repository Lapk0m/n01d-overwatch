Real-time Middle East and Global Conflict Intelligence Dashboard.
OSINT aggregation, flight tracking, maritime awareness, war ops monitoring, missile defense mapping, equipment order of battle, warzone radio streams, eclipse warfare analysis, satellite thermal detection, and war economy market impact tracking.

**No API keys required.** Uses publicly available OSINT feeds, the OpenSky Network, and NASA FIRMS satellite data.

---

## What is N01D Overwatch?

A Windows desktop intelligence dashboard that aggregates open-source intelligence (OSINT) from dozens of real-time sources into a single unified command interface. Built for researchers, analysts, journalists, and anyone tracking geopolitical events in conflict zones.

Think of it as a personal situation awareness terminal ‚ÄĒ pulling live data from RSS feeds, flight trackers, ship transponders, missile site databases, equipment deployments, warzone press radio streams, war operations monitoring, NASA satellite thermal hotspots, and conflict-driven market intelligence ‚ÄĒ all rendered on an interactive dark-themed map with XChat IRC aesthetics.

## Core Capabilities

### Multi-Source Intelligence Aggregation

- **RSS/OSINT Feeds** ‚ÄĒ 82+ sources including Reuters, AP, BBC, Al Jazeera, CENTCOM, Bellingcat, Janes, Liveuamap, The War Zone, ACLED, Airwars
- **Flight Tracking** ‚ÄĒ 160+ military callsign prefixes identified via OpenSky Network (drones, tankers, AWACS, fighters, SOF, aggressors)
- **War Ops Monitoring** ‚ÄĒ 34+ dedicated feeds across SIGACT, Sanctions, Cyber Operations, and Proxy/Militia activity
- **Missile/Defense Sites** ‚ÄĒ 24 tracked installations (IRGC ballistic missiles, Iron Dome, Arrow-2/3, THAAD, S-300/400, Patriot)
- **Equipment OOB** ‚ÄĒ Order of Battle for 8 regional forces with ~60 weapon systems, deployment locations, active counts
- **Warzone Radio** ‚ÄĒ Live press radio streams from conflict zones and CENTCOM briefings
- **Eclipse Warfare** ‚ÄĒ 11 events (2025-2028) with military significance analysis
- **Maritime Awareness** ‚ÄĒ AIS ship tracking in Strait of Hormuz, Red Sea, Persian Gulf
- **Ground Tracking** ‚ÄĒ NASA FIRMS VIIRS satellite thermal hotspots auto-refreshed every 10 minutes across the Middle East conflict zone
- **War Economy / Market Impacts** ‚ÄĒ 9 market sectors tracked: Energy, Defense, Commodities, Currencies, Shipping, Cybersecurity, Reconstruction, Insurance, Crypto. Commodity prices, defense stock watchlist, sanctions tracker, and trade feed aggregator
- **Equipment Intel Scanner** ‚ÄĒ 12 defense RSS feeds (Defense News, Janes, The War Zone, Breaking Defense, Naval News, etc.) scanned for procurement, delivery, and deployment keywords
- **Live Tracking** ‚ÄĒ 30-second auto-polling with real-time LIVE status indicator
- **Auto-Refresh Pipeline** ‚ÄĒ RSS/OSINT (configurable), Ships/Flights/WarOps (5 min), NASA FIRMS thermal (10 min), Equipment Intel RSS (every refresh cycle)

### War Ops ‚ÄĒ 4-Category Threat Monitor

- **SIGACT** ‚ÄĒ Drone strikes, airstrikes, artillery, ground ops, naval engagements, airspace closures. Feeds: CENTCOM, Liveuamap, Bellingcat, The War Zone, ACLED, Airwars, SOFREP
- **SANCTIONS** ‚ÄĒ OFAC designations, arms embargos, weapons sales, IAEA inspections, UN resolutions. Feeds: SIPRI, US Treasury, EU Sanctions Map, Arms Control Assoc., IAEA
- **CYBER** ‚ÄĒ APT groups, infrastructure attacks, state-sponsored hacking, electronic warfare. Feeds: The Record, Mandiant, CyberScoop, Dark Reading, Krebs on Security
- **PROXY** ‚ÄĒ Hezbollah, Houthi, PMF, Hamas, ISIS, SDF operations and movements. Feeds: Al Monitor, Middle East Eye, Iran International, Jamestown Foundation

- JS-to-C# message bridge for real-time updates
- Heatmap overlay showing conflict density

### Prerequisites

- Windows 10/11 (64-bit)
- .NET 8.0 SDK ‚ÄĒ https://dotnet.microsoft.com/download/dotnet/8.0
- WebView2 Runtime (included with modern Windows / Edge)

### Build and Run

```powershell
git clone https://github.com/bad-antics/n01d-overwatch.git
cd n01d-overwatch
dotnet build
dotnet run
```

## Severity Classification

Events are automatically classified into 4 severity levels:

- **CRITICAL** (Red) ‚ÄĒ Nuclear events, chemical attacks, invasion, assassination, WMD
- **HIGH** (Orange) ‚ÄĒ Airstrikes, missile strikes, casualties, APT zero-day, sanctions
- **MEDIUM** (Yellow) ‚ÄĒ Notable military movements, diplomatic developments
- **LOW** (Green) ‚ÄĒ Routine monitoring, background intelligence
